    WorldWind Explorer

    The WorldWind Explorer was created by Bruce Schubert and is powered by ESA-NASA Web WorldWind which provides the 3D Earth visualization with WMS maps, imagery and terrain. You are encouraged to use the Explorer as an online resource or use its source code to build your own 3D globe application. The User Guide and developer documentation are available online.

    Open Source Libraries
    • ESA-NASA WorldWind provides the 3D virtual globe
    • KnockoutJS provides the Model-View-View Model (MVVM) architecture
    • Bootstrap provides the responsive styling
    • JQuery is used to dynamically load DOM view fragments
    • JQuery UI adds UI elements
    • Dragula adds drag-n-drop support
    • SplitJS adds window splitters to the main layout
    • Moment provides date/time parsing and manipulation
    • RequireJS is a JavaScript file and module loader used improve the speed and quality of the code
